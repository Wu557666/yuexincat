## 功能概述

一个桌宠小猫，支持多只小猫播放，判断逻辑见main.py

## 使用方法
1. 安装依赖见requirement.txt
2. 启动桌宠小猫运行main.py

3. 右键小猫有四个选项{互动，形象管理，自动漫游（自动换装），退出}

4. 支持多个小猫，文件开头GIF_ASSETS支持添加更多小猫，当然你得有他们的文件

5. 文件用pyinstaller打包为exe，Windows可以直接运行看看效果
